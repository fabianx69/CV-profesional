# Multilingual Professional Portfolio 🌍

Este es el repositorio de mi currículum profesional web, diseñado con un enfoque minimalista y optimizado para ser consultado en múltiples idiomas, inspirado en el formato clásico **EUR** de las consolas portátiles y expandido para el mercado global.

## 🚀 Tecnologías
* **HTML5 / CSS3** (Estructura semántica)
* **Tailwind CSS** (Styling moderno y responsivo)
* **Vanilla JavaScript** (Lógica de internacionalización i18n sin dependencias pesadas)

## 🌐 Idiomas Soportados
* **Región EUR:** Español, Inglés (UK/US), Francés, Alemán, Italiano.
* **Global:** Chino (Mandarín), Coreano, Japonés, Hindi.

## 🛠️ Arquitectura y Mejores Prácticas
El proyecto sigue principios de **Clean Code** y está preparado para despliegues rápidos en arquitecturas Cloud (GCP/Firebase/GitHub Pages). 
* **SecDevOps Ready:** Estructura modular para fácil integración de auditorías de seguridad.
* **Performance:** Carga ultrarrápida al no utilizar frameworks pesados de JS.
